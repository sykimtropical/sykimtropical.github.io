인터넷이 자주 끊긴다던지 IP 관련 문제(충돌 등)가 생기는 것 같을 경우 아래 명령어 수행 가능

IP 초기화
```cmd
ipconfig /release
```

IP 갱신
```cmd
ipconfig /renew
```
